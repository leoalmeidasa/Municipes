## Setup

1. `git clone https://github.com/leoalmeidasa/Municipes`
2. `cd Municipes`
3. `rails db:create db:migrate`
4. `rspec spec/` (para verificar se todos os testes estão ok)
5. `rails s`
6. Abrir o link localhost:3000
